Browsers

Microsoft Edge Home Page

Microsoft's home page for its Microsoft Edge browser, designed as the replacement for Microsoft Explorer. Includes information about Microsoft Edge and download links.

https://www.microsoft.com/en-us/edge

Microsoft Edge Help Page

Links to further information about Microsoft Edge.

https://support.microsoft.com/en-us/microsoft-edge

Internet Explorer Home Page

Microsoft's home page for its Web browser, Internet Explorer. Includes download links for Internet Explorer 11, as well as security updates for previous versions of Internet Explorer. Note that Microsoft will retire Internet Explorer on June 15, 2022.

https://www.microsoft.com/en-us/download/internet-explorer.aspx

Internet Explorer Help Page

Links to a variety of FAQs.

https://support.microsoft.com/en-us/products/internet-explorer

Mozilla Firefox Home Page

Mozilla's home page for the Firefox Web browser. Includes download link for the latest version of Firefox.

https://www.mozilla.org/en-US/firefox

Firefox Support Home Page

Links to Help and tutorials.

https://support.mozilla.org/en-US

Apple Safari

Apple's home page for its Safari Web browser.

https://www.apple.com/safari

Google Chrome

Home page for the Google Chrome browser.

https://www.google.com/chrome
